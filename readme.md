## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/awahmadwahab/aw_portfolio-1.git
    ```
2. Navigate to the project directory:
    ```bash
    cd aw_portfolio-1
    ```

## Usage

Open the `index.html` file in your web browser to view the portfolio website.
